# Gladiator.tf-Item-statistics-button
Improves item liquidity analytics.
# Installation
1. Install <a href="https://www.tampermonkey.net/" target="_blank" rel="noopener noreferrer">Tampermonkey</a> browser extension
2. Click <a href="https://github.com/mrTranzister/Gladiator.tf-Item-statistics-button/blob/main/Gladiator.tf%20%E2%80%93%20Item%20statistics%20button-1.0.user.js" target="_blank" rel="noopener noreferrer"> here to install the script</a> or copy the code manually
3. This script automatically runs on the Gladiator page in the items section.
# Features
This script adds a button that opens a new tab with the item’s statistics.
